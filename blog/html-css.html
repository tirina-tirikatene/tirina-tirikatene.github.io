The differences between Margin, Border and Padding

Lets look at how and when to use Margin, Border and Padding along with the differences.

  • Border
    • A border is a visible line around an element. Borders can be used to visually seperate content, create a design aspect on the webpage or even highlight elements to make them stand out.]
  • Margin
    • A margin is a fundamental part of CSS as it controls the layout and spacing of elements. The margin defines the space outside the border. They are used to ensure content is not too close together. Margins are not included in the clickable area and dont have a background colour.
  • Padding
    • Padding is used to ensure the content is visually appealing and that it does not touch the element's border. It is the space between the content of an element and its border. Padding creates space inside the border. It makes sure that text des not directly touch the border which in turn improves the aesthetics and ensures we can read it easily.
