An analogy that describes JavaScript and its relationship to HTML and CSS.

We are rocking a house building analogy.

So, let’s say we are building a house.

  • HTML would be like the structural components. It is the foundation and walls of the house that hold it all together.
  • CSS however is the superficial part. Its everything you might use to decorate the house, paint, carpet, maybe some wallpaper.
  • JavaScript is then the part that gives the house functionality. Things like the TV, oven, dishwasher.

Explain control flow and loops using an example process from everyday life.

Today we are going to make some school lunches.

Control flow (from what I understand) is following a sequence of steps which might look like this-

  1. Make the decision to make lunches
  2. Open lunchbox
  3. Collect items
  4. Add items to lunchbox
  5. Put items away
  6. Close lid
  7. Pack lunchbox into bag

Whereas if I was making lunches in a loop:

  1. Start making lunches
  2. Check how many I have made, have I made enough?
  • Yes, exit the loop
  • No, continue filling next lunchbox
  • Once I have made the required number, stop.

Control flow organises the steps needed and the loop makes sure the process is repeated a certain number of times.

Describe what the DOM is and an example of how you might interact with it.

DOM stands for Document Object Model.

It lets programming tools like JavaScript play with the HTML and CSS of the webpage, it enables user interactions. We can access the DOM via web console. The DOM is then used to access the document and all its elements using JavaScript.

If we were to go back to the building house analogy, the DOM is almost like the blueprint of a house. It tells you where each room is and what’s inside. We are able to access the blueprint which tells us where each piece of furniture is and then use JavaScript to change the colour of some of the furniture or even add new pieces of furniture.

Explain the difference between accessing data from arrays and objects.

Arrays use positions/numbers to find items.

For example, heres a list of colors. 

    let colors = ["red", "green", "blue"];

To access each colour you would put:

  • colors[0] gives you "red" (the first color).
  • colors[1] gives you "green" (the second color).
  • and so on

Objects use labels/names to find items.

Now heres a profile of a person

    let profile = {
        name: "John",
        age: 25,
        city: "Los Angeles"
      };
Now Here is how you would access each individual piece of information:
  • profile.name gives you "John" (the person's name)
  • profile.age gives you 25 (the person's age)
  • profile.city gives you "Los Angeles" (the city where the person lives)

Explain what functions are and why they are helpful.

Functions are a reusable set of instructions that help perform a specific task each time without having to remember the steps. You are able to use the same function anytime its needed without having to rewrite it.

Functions are very helpful as they save time and effort. They help simplify tasks and avoid the need to repeat code which also keeps everything much cleaner and less confusing.
